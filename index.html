Sobre:

Sou desenvolvedor fullstack, atualmente cursando Desenvolvimento de Sistemas na ETEC Zona Leste e o curso de Desenvolvimento Java FullStack pela Generation Brazil. Minha trajetória é guiada pela curiosidade e pelo desejo de usar a tecnologia para resolver problemas reais, explorando novas abordagens para aprimorar tanto minhas habilidades técnicas quanto minhas competências interpessoais.

Principais realizações: Meu principal projeto é a Rede Conectados, uma plataforma colaborativa e gamificada desenvolvida como Trabalho de Conclusão de Curso (TCC). Inspirado na Pirâmide de Aprendizagem de William Glasser, o projeto fortalece o aprendizado entre estudantes, recompensando aqueles que mais contribuem para o crescimento dos colegas. Esta experiência me permitiu aplicar e aprimorar meus conhecimentos em Angular, Node.js, MySQL, APIs REST e ORM, além de desenvolver habilidades de gestão de projetos e inovação.

O que me motiva: Amo explorar novas soluções tecnológicas e trabalhar em equipe para criar software que realmente ajude as pessoas. Estou sempre em busca de desafios que me permitam aprender e crescer, contribuindo para projetos com impacto positivo.
